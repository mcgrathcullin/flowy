Flowy Guide

Basic Syntax

Type: Text

Example: Start: Begin Process

Node Types

  • Start: Beginning of the flow
  • Block: Process or action step
  • Tree: Decision point
  • Note: Additional information
  • Label: Decision outcome
  • Number: Branch destination (1:, 2:, etc.)

Indentation

Use > to create hierarchy:

Block: Parent
>Block: Child
>>Block: Grandchild

Decision Trees

Tree: Quality Check
>Label: Pass
>>1: Move to Storage
>Label: Fail
>>2: Return Item
